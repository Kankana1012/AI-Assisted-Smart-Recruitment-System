
## 📊 Dataset

The dataset used in this project is publicly available on Kaggle and is **not included in this repository** due to GitHub file size limitations.

**Dataset:** Resume Dataset by Snehaan Bhawal  
**Download:** https://www.kaggle.com/datasets/snehaanbhawal/resume-dataset

This dataset contains over **2,400 resumes** in **PDF** and **text** formats spanning **24 job categories**. It is used for resume parsing, information extraction, text classification, and recruitment-related NLP tasks.

> **Note:** Please download the dataset from Kaggle and place it in the appropriate project directory before running the code.
